Documentação Operacional

Como usar as telas (Passo a Passo)

Para utilizar o StandardPoint corretamente, siga o fluxo de cadastro e análise nesta ordem:

  1. Empresa: Cadastre a empresa alvo da análise.
  2. Projeto: Crie um projeto vinculado à empresa cadastrada.
  3. Requisitos: Importante! Você precisa cadastrar os requisitos para poder associar as funções posteriormente.
  4. Funções: Realize a contagem e classificação das funções de dados (ALI, AIE) e transação (EE, SE, CE), vinculando-as aos requisitos pertinentes.
  5. Relatórios: Acesse a área de relatórios para visualizar os totais e análises do projeto.

Valor de Ajuste de Função (VAF)

Aviso: É necessário preencher o questionário do VAF (Valor de Ajuste de Função) nas configurações do projeto para que o cálculo final fique de acordo com as normas do IFPUG. Sem isso, apenas o PFNA (Pontos de Função Não Ajustados) será exibido com precisão.

Tipos de Funções

  • Desenvolvimento: Usado na criação de um novo software. Todas as funções são avaliadas.
  • Melhoria: Usado quando o software já existe e passará por alterações (Adição, Alteração ou Exclusão).
  • Baseline: Contagem do tamanho funcional de um sistema existente, refletindo seu estado atual.

Gerenciamento do Sistema

Salvar um projeto

O aplicativo gerencia os dados e tem auto-save, mas você também pode realizar backups nas configurações ou quando solicitado. Certifique-se de salvar todas as modificações no formulário.

Exportar para PDF

Dentro da tela de Relatórios, utilize o botão "Exportar PDF" para gerar um arquivo detalhado contendo a contagem e métricas do projeto selecionado.

Configurar usuários

Acesse o menu "Configurações" (ou gerencie os acessos/autenticação no fluxo inicial) para criar e gerenciar perfis de analistas de métricas.

Dica: Use a tecla Esc para fechar rapidamente.
